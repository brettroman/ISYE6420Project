====================================
Setting Up the Environment
====================================
1. Clone the repository:
   git clone https://github.com/brettroman/ISYE6420Project.git

2. Navigate to the project directory:
   cd ISYE6420Project

3. Install dependencies:
   pip install -r requirements.txt


====================================
Run data collection and plotting
====================================

1. `py src/retrieve_data.py`
2. `py src/plot_data.py`



====================================
View Plots
====================================

- Directory: `visualizations/`
